# 🔓 VLESS подписка — только белые подсети РФ

Автоматически обновляемая VLESS подписка, содержащая **исключительно** серверы из белых подсетей РФ (VK, Яндекс, Selectel, Timeweb, Aeza, МТС, Билайн и др.).
Работает при включённых белых списках у операторов (МТС, Теле2, Мегафон, Билайн).
Обновляется каждый час через GitHub Actions.

## Подписка

**Ссылка для v2rayTUN / v2rayNG / Shadowrocket / Karing:**

```
https://raw.githubusercontent.com/J-L33T/vlesstj/main/vless.txt
```

## Как работает

1. Скачивает VLESS конфиги из проверенных публичных источников
2. 2. Фильтрует — оставляет только серверы с IP из белых подсетей РФ
   3. 3. Проверяет каждый живой сервер — TCP connect + TLS handshake
      4. 4. Мёртвые серверы отсеиваются автоматически
         5. 5. Итоговый список **всех живых белых серверов** пушится в репо
           
            6. ## Пометки серверов
           
            7. | Пометка | Значение |
            8. |---------|----------|
            9. | 🇷🇺 WL | IP из белых подсетей РФ — гарантированно работает при белых списках |
