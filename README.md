# 🔓 VLESS подписка — обход белых списков РФ

Автоматически обновляемая VLESS подписка для обхода ограничений мобильного интернета в России (МТС, Теле2, Мегафон, Билайн).
Обновляется каждый час через GitHub Actions.

## Подписка

**Ссылка для v2rayTUN / v2rayNG / Shadowrocket / Karing:**
```
https://raw.githubusercontent.com/J-L33T/vlesstj/main/vless.txt
```

## Как работает

1. Скачивает VLESS конфиги из проверенных публичных источников
2. Проверяет каждый сервер — TCP connect + TLS handshake
3. Мёртвые серверы отсеиваются автоматически
4. Живые серверы сортируются — сначала IP из белых подсетей РФ
5. Итоговый список из 100 серверов пушится в репо

## Пометки серверов

| Пометка | Значение |
|---------|----------|
| `🇷🇺 WL` | IP из белых подсетей РФ (VK, Яндекс, Selectel и др.) — приоритет при белых списках |
| `🌍` | Внешний сервер — работает в обычных условиях |

> При включении белых списков оператором — пробуй серверы с `🇷🇺 WL` в первую очередь.

## Источники

Подписка агрегирует конфиги из следующих публичных репозиториев:

| Автор | Репозиторий | Описание |
|-------|-------------|----------|
| [@zieng2](https://github.com/zieng2) | [zieng2/wl](https://github.com/zieng2/wl) | Подписка специально для обхода белых списков, тестируется на реальных симках |
| [@igareck](https://github.com/igareck) | [igareck/vpn-configs-for-russia](https://github.com/igareck/vpn-configs-for-russia) | Коллекция VLESS конфигов для РФ, CIDR и SNI подписки для мобильного интернета |
| [@whoahaow](https://github.com/whoahaow) | [whoahaow/rjsxrd](https://github.com/whoahaow/rjsxrd) | Верифицированные конфиги с проверкой через Xray-core |
| [@kort0881](https://github.com/kort0881) | [kort0881/vpn-vless-configs-russia](https://github.com/kort0881/vpn-vless-configs-russia) | Большая коллекция VLESS/VMESS/Shadowsocks для России |

Все права на исходные конфиги принадлежат их авторам.
Данный репозиторий только агрегирует и фильтрует публично доступные данные.
