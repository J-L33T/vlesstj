# 🔓 VLESS подписка — обход белых списков МТС/Теле2/Мегафон/Билайн

Автоматически обновляемая подписка VLESS Reality для обхода ограничений мобильного интернета в РФ.
Обновляется каждый час через GitHub Actions.

## Как работает

Скрипт собирает конфиги из проверенных публичных источников и применяет двойную фильтрацию:

1. **Только Reality** — протокол `security=reality`, маскируется под легитимный TLS трафик
2. **Белый IP** — адрес сервера принадлежит российскому или пропускаемому хостеру (Selectel, Яндекс Облако, Hetzner и др.) — определяется по ASN через RIPE API
3. **Белый SNI** — домен в параметре `sni=` оканчивается на `.ru` или входит в список доменов которые пропускают операторы
4. **Дедупликация** — один и тот же сервер из разных источников попадает в подписку только один раз

Итоговый список содержит до 100 серверов, все с пометкой 🇷🇺 WL.

## Почему именно такой подход

С 2025 года операторы РФ при ограничениях применяют **комбинированную проверку IP + SNI**:
- IP сервера должен быть из подсетей разрешённых провайдеров
- SNI (домен в TLS handshake) должен быть российским

Простой SNI-спуфинг без белого IP уже не работает на МТС/Теле2.
Обычные VPN (WireGuard, OpenVPN, Shadowsocks) блокируются на уровне DPI.

## ⚠️ Безопасность

Все конфиги **публичные** — UUID и ключи видны всем. Владелец сервера видит твой трафик.

- ✅ Подходит для обхода ограничений в моменты блокировок
- ❌ Не подходит для банкинга, личной переписки и рабочих данных

Для приватного использования — поднимите собственный сервер на российском IP (Selectel, Яндекс Облако).
